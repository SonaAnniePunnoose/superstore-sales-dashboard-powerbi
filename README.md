# Superstore Sales Performance Dashboard (Power BI)
## Project Overview
This project analyzes Superstore sales data to understand sales performance across product categories, regions, and customer segments.
An interactive Power BI dashboard was built to visualize key business insights.

## Tools Used
* Power BI
* Data Visualization
* Data Cleaning
* DAX

## Dataset
The dataset contains sales transactions including:
* Order ID
* Order Date
* Customer Name
* Category
* Region
* Segment
* Product Name
* Sales

## Key KPIs
* Total Orders: 4922
* Total Sales: 2.26M
* Total Customers: 793
* Average Sales: 230.77

## Dashboard Insights
* Technology category generates the highest sales.
* West region contributes the largest share of sales.
* Consumer segment accounts for the majority of customers.
* Sales show monthly fluctuations indicating seasonal demand.
